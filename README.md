## 项目简介

这是一套**自动化内容生成系统**，通过抓取爆款公众号文章，分析规律，提炼写作手册，自动生成高质量公众号文章，经AI审核+自动重写+润色后推送通知。

**核心流程：**
```
抓取爆款文章 → 提取写作规律 → 提炼写作手册 → 生成初稿(Claude) → Coze审核 → 不通过自动重写(最多3轮) → 通过后润色(编辑手册) → Server酱通知 → 人工发布到公众号
```

## 人设与内容定位

### 主角人设
- 33岁男性，自由职业，具体做什么不说
- 深圳有个租的小房子，但经常不在
- 经历过：辞职、创业、失败、赚到过钱
- 不焦虑但也没多洒脱，有点懒，有点嘴硬
- 对物质态度松弛——小镇出来的，现在买得起好东西，但不炫也不装

### 固定配角（不是每篇标配，相关才出场）
- 做餐饮的老朋友（男），说话很直
- 一个"她"，身份模糊
- 妈妈，在老家（注意：不要总写她打电话问瘦没瘦！）
- 比自己小几岁的后辈
- 路上遇到的陌生人

### 生活阶段（按月份，仅做参考不强制）
- 1-2月：深圳自己的小房子
- 3-4月：南方小城慢住
- 5-6月：东南亚
- 7-8月：回老家
- 9-10月：北方某城市
- 11-12月：回深圳

### 内容风格目标
- **对标S叔（Spenser）**：用自己的经历讲道理，敢说、锐利、自嘲、真诚
- 不是散文，不是鸡汤，是"一个经历过事的人跟你喝酒说的真话"
- 每篇要有一个读者能拿走的观点或认知
- 读完感受应该是"被点醒了"而不是"被治愈了"

## 选题系统

### 创作方向（带权重）
- **70%概率**抽到高价值方向：认知与观点、商业与人性、选择与取舍、反常识、花钱的智慧、人际洞察、赚钱弯路、反驳热门观点
- **30%概率**抽到普通方向：食物、人、地方、物件、变化、日常碎片、回忆、关系

### 防重复机制
- 每次生成前从Airtable拉最近5篇文章标题，告诉Claude"这些写过了不要重复"
- 铁律中明确禁止重复桥段（妈妈打电话、煮面、植物比喻等）

### 指定主题
- 手动触发daily.yml时可输入指定主题
- 指定主题时跳过人设限制，用合适的视角写

## Coze Bot说明

目前只有一个Bot，同时承担审核和灵感扩写两个功能：
- 消息以"审核"开头 → 执行文章审核打分，返回JSON
- 消息以"写文章"开头 → 执行灵感扩写
- 其他消息 → 普通聊天回复

Bot发布到两个渠道：
- API → 供review.py调用审核
- 微信订阅号 → 供灵感扩写

**注意：COZE_API_TOKEN有效期1个月，过期需要重新生成并更新GitHub Secrets！**

## 已知问题与解决方案

### 1. 表名问题（重要！）
- `爆款文章库` — 中文名
- `patterns`、`prompts`、`contents` — 英文名
- 混用会导致403错误

### 2. contents表字段名
- 以前叫`状态`，现在叫`status`
- 代码中contents相关的地方全部用`status`
- 其他表（爆款文章库、prompts）仍然用`状态`

### 3. Coze API流式响应
- 必须用`stream: True`
- 从SSE流式响应中拼接answer内容：`answer += content`（拼接不是覆盖）
- 需要过滤非dict的event和`[DONE]`标记

### 4. Coze Bot回复格式
- Bot有时返回JSON后面带多余文字
- `parse_json_response`用花括号匹配提取第一个完整JSON对象

### 5. 编码问题
- Airtable headers需要`Content-Type: application/json; charset=utf-8`
- Coze流式响应需要`resp.encoding = "utf-8"`

### 6. Prompt版本管理
- 改完`prompt_builder.py`后，需要去Airtable把当前版本的Prompt状态改为`历史版本`
- 这样`main.py generate`找不到当前版本时，会自动用新代码重新构建

## 内容质量迭代历程

供新Claude理解用户的审美标准：

1. **初始阶段**：千问生成，教科书式，"第一步第二步第三步"
2. **加入写作手册**：有范文参考，但还是一板一眼
3. **加入人设**：有生活气，但文艺范太重，全是煮面和发呆
4. **修正"装酷"**：禁止"我没接话"连发，加入正常人互动
5. **提升价值感**：从"被治愈"到"被点醒"，加入认知类选题
6. **切换Claude**：千问→Claude Sonnet 4.6，质量大幅提升
7. **编辑手册润色**：S叔风格润色标准，传播力提升
8. **当前状态**：能稳定输出可发布文章，有价值感和传播力

### 用户的核心审美标准（重要！）
- **不装**：不沉默装深沉，不用诗意逃避问题
- **敢说**：有明确观点，敢下判断，不含糊
- **真实**：敢暴露自己不好看的一面
- **有用**：读完能拿走一个观点或认知
- **不说教**：用"我发现""我后来才懂"的语气
- **传播力**：要有让人想截图转发的句子
- **对标S叔**：像一个经历过事的人跟你喝酒说的真话
- **最讨厌的**：妈妈打电话问瘦没瘦、煮面结尾、植物比喻人生、"我没接话"连发三次、环境描写堆砌、碎片拼盘讲五个故事、观点含糊不敢下判断
